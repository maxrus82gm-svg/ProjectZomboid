Короткий список рабочих правил проекта `Project Zomboid Dedicated Server`.

## Рабочие правила

- `Documentation/Правила_Git_и_Review.md`
- `Documentation/Правила_создания_документов.md`

## Главное правило

Перед задачей агент читает:

```text
M:\GitHub\ProjectZomboid\ProjectZomboud\Старт.md
M:\GitHub\ProjectZomboid\ProjectZomboud\Задача_агенту.md
```

Затем агент читает только те рабочие правила, которые нужны текущей задаче.
