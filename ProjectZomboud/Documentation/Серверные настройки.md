# Серверные настройки

Памятка для человека: где в Project Zomboid Dedicated Server прописывать моды, стартовые предметы и что именно менять в файлах.

## Связи Obsidian

- [[Старт]]
- [[Правила_сервера_Project_Zomboid]]

## Самое важное

- Моды подключаются в `servertest.ini`.
- Стартовые предметы персонажа настраиваются не в списке модов.
- Перед любой правкой серверного конфига сначала сделать backup рядом с файлом.
- Если в конфиге есть русские комментарии, сохранить файл в исходной кодировке. Для текущего сервера это может быть Windows-1251.
- `servertest_SandboxVars.lua` нельзя пересохранять в UTF-8 с BOM. Сервер может упасть с ошибкой Lua при запуске. Менять лучше точечно и сохранять исходную кодировку файла.
- Уже созданные персонажи обычно не получают новые стартовые предметы. Проверять нужно на новом персонаже после применения настроек.

## Где смотреть текущие стартовые настройки

Текущие стартовые настройки сервера смотреть в папке:

```text
\\192.168.0.77\Zomboid\Server
```

Стартовые предметы и подключение модов смотреть в:

```text
\\192.168.0.77\Zomboid\Server\servertest.ini
```

Важные строки:

```text
SpawnItems=
Mods=
WorkshopItems=
```

`SpawnItems=` показывает ручной список предметов, которые выдаются новому персонажу при спавне.

`Mods=` показывает, какие ModID включены на сервере.

`WorkshopItems=` показывает, какие Steam Workshop ID подключены к серверу.

Стартовый набор, вода и электричество смотрятся в:

```text
\\192.168.0.77\Zomboid\Server\servertest_SandboxVars.lua
```

Важные строки:

```lua
StarterKit = false,
WaterShutModifier = 55,
ElecShutModifier = 35,
```

`StarterKit` включает или выключает стандартный стартовый набор.

`WaterShutModifier` задаёт день отключения воды.

`ElecShutModifier` задаёт день отключения электричества.

## Где прописывать моды

Моды прописываются в:

```text
servertest.ini
```

Нужные строки:

```text
WorkshopItems=...
Mods=...
```

`WorkshopItems=` - это цифровые Steam Workshop ID. Они пишутся через `;`.

Пример:

```text
WorkshopItems=3455290347;3748865313
```

`Mods=` - это ModID из файла `mod.info`. Они тоже пишутся через `;`.

Пример:

```text
Mods=PZB42RUTranslation;[B42] Project zomboid translate to Russian (by rennocki)
```

Готовый актуальный список для копирования хранится здесь:

```text
M:\GitHub\ProjectZomboid\ProjectZomboud\Documentation\Список_модов_Project_Zomboid.md
```

В строки `WorkshopItems=` и `Mods=` нельзя писать предметы вроде биты или рюкзака. Эти строки только для модов.

## Где лежат файлы модов

Локальный источник скачанных Workshop-модов:

```text
G:\SteamLibrary\steamapps\workshop\content\108600
```

Целевая Workshop-папка сервера:

```text
\\192.168.0.77\Server\Project Zomboid Dedicated Server\steamapps\workshop\content\108600
```

Workshop-моды копируются на сервер только в эту папку. Не класть моды просто рядом с сервером.

## Как выдать при спавне биту и школьный рюкзак

Есть два способа.

### Способ 1. Стартовый набор

Это самый простой вариант, если подходит стандартный набор игры.

Править нужно:

```text
servertest_SandboxVars.lua
```

Найти строку:

```lua
StarterKit = false,
```

Заменить на:

```lua
StarterKit = true,
```

Что выдаёт `StarterKit = true`:

- школьный рюкзак;
- бейсбольную биту;
- молоток;
- бутылку воды;
- чипсы.

В этом варианте рюкзак надевается на спину, а бита кладётся внутрь рюкзака.

### Способ 2. Только нужные предметы через SpawnItems

Этот вариант нужен, если хочется выдать именно биту и школьный рюкзак без стандартного набора.

Править нужно:

```text
servertest.ini
```

Найти строку:

```text
SpawnItems=
```

Записать так:

```text
SpawnItems=Base.BaseballBat,Base.Bag_Schoolbag
```

Важно:

- предметы в `SpawnItems=` пишутся через запятую;
- точка с запятой здесь не используется;
- `Base.BaseballBat` - бейсбольная бита;
- `Base.Bag_Schoolbag` - школьный рюкзак;
- предметы выдаются новым игрокам в инвентарь;
- рюкзак через `SpawnItems=` может не надеться автоматически на спину.

## Что выбрать для биты и школьного рюкзака

Если нужен обычный стартовый набор с рюкзаком на спине, выбрать:

```lua
StarterKit = true,
```

Если нужны только два предмета и без лишней воды, чипсов и молотка, выбрать:

```text
SpawnItems=Base.BaseballBat,Base.Bag_Schoolbag
```

## Может ли агент добавить это сам

Да, агент может сам внести эту настройку в серверные файлы, но только по отдельной явной команде владельца сервера.

Для включения стандартного стартового набора агент меняет:

```text
servertest_SandboxVars.lua
```

Строка должна стать такой:

```lua
StarterKit = true,
```

Для выдачи только биты и школьного рюкзака агент меняет:

```text
servertest.ini
```

Строка должна стать такой:

```text
SpawnItems=Base.BaseballBat,Base.Bag_Schoolbag
```

Перед правкой агент обязан:

- проверить, что нужный серверный файл доступен;
- сделать backup рядом с исходным файлом;
- изменить только нужную строку;
- сохранить исходную кодировку файла;
- сообщить, какой файл был изменён и какой backup создан.

Агент не должен сам запускать, останавливать или перезапускать сервер без отдельной явной команды.

## После изменения

1. Сохранить backup исходного файла рядом с конфигом.
2. Изменить только нужную строку.
3. Проверить, что кодировка файла не сломалась.
4. Перезапустить сервер, когда можно безопасно перезапускать.
5. Создать нового персонажа для проверки.

Сервер не запускать и не останавливать без отдельной явной команды владельца сервера.
